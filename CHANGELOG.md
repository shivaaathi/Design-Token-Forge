# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] — 2026-05-05

### Added
- 23 production-ready L1 components with full variable coverage (Button, Input, Toggle, Checkbox, Radio, Select, Textarea, Slider, DatePicker, FileUpload, Alert, Avatar, Badge, Tooltip, Toast, ProgressBar, ProgressCircle, IconButton, SplitButton, MenuButton, Editor, ColorSystem, ColorGenerator)
- 740+ color tokens across 4 tiers (Primitives → Semantic → Adaptive Surfaces → Component)
- 10 density size modes per component (2xs through 5xl)
- Dark/light theme support with automatic adaptive surfaces
- Interactive demo site with live variant switching
- Color System Generator with palette engine
- Token Editor with GitHub commit integration
- Framework integration snippets (React, Vue, Svelte)
- GitHub Pages deployment via CI/CD
- Visual regression testing with Playwright
- Multi-project token support (desktop-pdf-editor, writer-mobile, writer-handhelds)
