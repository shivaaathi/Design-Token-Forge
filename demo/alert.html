Live

All four semantic roles with icon, title, body, and accent bar. Dismiss to remove.

Semantic Roles

Surface

Info (default), success, warning, danger. Each role maps to a unique bg, fg, accent, and icon color.

Variants

Surface

Default (soft container bg), filled (solid role bg), outline (transparent + border). Shown for each role.

Sizes

Dimension

Three density tiers — small, base, large. Padding scales proportionally.

Accent Bar

Shape

Left-border accent pattern with rounded cap. Uses --alert-accent-width and role-specific --alert-*-accent colors.

Dismissible

Interactive

Click × to dismiss. Alert animates out with opacity and translateY transition.

Accessibility

A11y

role="alert" for assertive live region (danger/warning). role="status" for polite (info/success).

ARIA Roles

Danger/warning use role="alert" (assertive). Info/success use role="status" (polite).

Forced Colors

High Contrast mode shows CanvasText border. Accent bar uses Highlight.

Close Button

Dismiss button includes aria-label="Dismiss" and visible focus ring.

Use in Your Framework

Integration

Wrap this component in your framework. Props become data attributes — our CSS does the rest.
