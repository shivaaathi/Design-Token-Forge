Tooltip Explorer — Design Token Forge

~60 CSS Variables 4 Variants 4 Placements 3 Sizes 7 Axes

Hero Preview

Live

Hover triggers to preview default, light, error, and success tooltip variants. Arrow points toward the trigger.

Resolved Tokens

Variants

Surface

Default (dark), light (bordered), error, and success. Each variant sets bg, fg, and arrow color tokens.

Placement

Layout

Four cardinal placements: top, bottom, left, right. Arrow auto-rotates per placement.

Sizes

Dimension

Three density tiers — small, base, large. Padding, font-size, max-width, and icon-size scale together.

Rich Content

Slots

Tooltip supports icon, title, and multi-line body slots for richer contextual help.

Accessibility

A11y

Tooltip uses role="tooltip", pairs with aria-describedby on trigger. Forced Colors shows visible border.

ARIA Pairing

Trigger has aria-describedby pointing to tooltip id. Tooltip has role="tooltip".

Forced Colors

High Contrast mode shows a 1px CanvasText border. Arrow also visible.

Reduced Motion

Scale/opacity animation collapses to instant when prefers-reduced-motion is set.

Use in Your Framework

Integration

Wrap this component in your framework. Props become data attributes — our CSS does the rest.
