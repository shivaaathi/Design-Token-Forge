Setup Instructions

  1. Create a new GitHub repository: my-design-tokens
  2. Extract the downloaded zip and push all files: 
    cd my-design-tokens
git init
git add .
git commit -m "Initial token setup"
git branch -M main
git remote add origin https://github.com/your-username/my-design-tokens.git
git push -u origin main
  3. Enable GitHub Pages in your repo:
    Settings → Pages → Source: GitHub Actions
  4. Wait for the first deploy (check Actions tab — takes ~1 minute)
  5. Open the Design Token Forge plugin in Figma
  6. Expand Server URL and enter: 
    https://your-username.github.io/my-design-tokens
  7. Click Save → The plugin connects and shows your tokens
  8. Click Create Variables to sync all tokens into your Figma file

Updating Tokens Later

  1. Edit config.json in your repo (change colors, mappings, etc.)
  2. Commit and push to main
  3. GitHub Actions rebuilds automatically (~1 min)
  4. The Figma plugin detects the change and shows "Token changes detected"
  5. Click Update Variables — done!
