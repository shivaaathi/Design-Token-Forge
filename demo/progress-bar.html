Role Colors

Surface

Primary (default), success, warning, danger, neutral. Fill color maps to the role's component-bg token.

Density Scale

Dimension

10 height modes from 2px (micro) to 24px (ultra). Track and fill radius auto-match.

Striped

Surface

Diagonal stripe overlay via CSS gradient. Animated version adds a moving stripe effect.

Indeterminate

Motion

Infinite sliding fill for unknown-duration operations. Width and speed are token-driven.

With Label

Slots

Optional percentage label sits beside the track. Font inherits from progress tokens.

Accessibility

A11y

role="progressbar" with aria-valuenow, aria-valuemin, aria-valuemax.

ARIA Attributes

Determinate: aria-valuenow. Indeterminate: omit aria-valuenow.

Forced Colors

Track gets CanvasText border. Fill uses Highlight.

Reduced Motion

Stripe and indeterminate animations stop. Static fill only.
